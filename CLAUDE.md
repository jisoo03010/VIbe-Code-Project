- 프로젝트 명 : Keep-It (꾸준함 관리 TODO 리스트)
- 문제 정의  : "조금씩 꾸준히"가 어려운 사용자를 위해 시각적인 성취감을 제공하고 할 일을 관리해주는 서비스

- 웹 형태 : 
1열 단일 레이아웃
버튼 크기: 44px × 44px
텍스트 크기: 14px 이상
패딩: 8px ~ 12px
줄 높이: 1.5em
가로 : 320~480px


1. UI/UX 디자인 가이드 
모바일 웹 환경에 최적화된 단일 레이아웃을 지향합니다.

- 레이아웃: 1열 단일 레이아웃 (Center Aligned)

- 사이즈 제약: 가로 폭 320px ~ 480px (Max-width 설정 필수)

- 타이포그래피: 기본 텍스트 14px 이상 / 줄 높이 1.5em

- 터치 요소: 모든 버튼 및 클릭 요소는 최소 44px × 44px 확보

- 간격: 내부 패딩 8px ~ 12px 유지

- 메인 컬러 팔레트:
#96ce46


2. 주요 기능 (Requirements)
[핵심] 캘린더 메인 화면
뷰 전환: 주간(Week) / 월간(Month) 토글 기능.
인터랙션: 날짜 클릭 시 해당 날짜의 TODO 리스트 하단 노출 또는 모달 팝업.
시각화: 할 일 완료율에 따라 날짜별 배경색 또는 인디케이터 표시.

[기능] 일정 관리 (CRUD)
Create: 할 일 추가 (내용, 목표 시간, 카테고리 선택).
Read: 일별 할 일 목록 조회.
Update: 내용 수정 및 완료 체크(Status 전환).
Delete: 일정 삭제.

- 폴더 구조 
기능 컴포넌트별로 폴더를 격리하여 관리합니다.



Keep-It 캘린더 헤더 리디자인프로젝트 컨텍스트Keep-It은 React Native 기반의 개인 일정/할일 관리 앱이다. 캘린더 헤더 영역을 리디자인해줘.디자인 컨셉: "콩심은데 콩나고"할일 달성률에 따라 커피콩 씨앗이 점점 자라나는 성장 일러스트를 헤더에 표시한다. 사용자가 할일을 많이 완료할수록 식물이 성장하여 동기부여를 주는 게이미피케이션 요소다.성장 5단계 (달성률 기준)
1단계 (0~20%) — 씨앗: 흙 위에 커피콩 씨앗이 놓여있는 모습
2단계 (21~40%) — 발아: 씨앗이 갈라지며 작은 뿌리가 내려가는 모습
3단계 (41~60%) — 새싹: 흙을 뚫고 두 잎의 새싹이 올라온 모습
4단계 (61~80%) — 줄기: 줄기가 자라고 잎이 여러 장 달린 모습
5단계 (81~100%) — 꽃/열매: 꽃이 피거나 커피 체리 열매가 열린 모습
각 단계는 이전 단계의 요소를 포함하면서 새로운 요소가 추가되는 누적 형태로 구현해줘.단계 전환 애니메이션
react-native-reanimated로 단계 전환 시 spring 애니메이션 적용
새 요소(잎, 꽃 등)는 scale 0→1로 팝업 등장
식물 전체에 살짝 흔들리는 swaying 미세 루프 애니메이션 추가
100% 달성 시 꽃/열매에 미세한 sparkle 이펙트


완료 개수	단계	모습
0~2개	씨앗	흙 위 커피콩
3~6개	발아	뿌리 + 싹
7~11개	새싹	줄기 + 두 잎
12~19개	줄기	긴 줄기 + 여러 잎
20개+	열매	커피 체리 + sparkle
30개+	보너스	열매 3개 추가 + sparkle 풍성
